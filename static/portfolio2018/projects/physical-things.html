PROJECTS — PHYSICAL THINGS

3.15.18

An Intelligent Window

Basements are artificially lit, dreary places. I find myself in a basement every week at Carnegie Mellon, in the IDeATe program, which resides in the cinder-blocked basement of Hunt Library. For the class Intelligent Environments, I'm building a window, designed to be installed indoors in a place that otherwise could not get natural light, such as a basement. The window will give the space access to the feel of outdoor conditions by way of a weather API and an array of LEDs. This project is a collaboration with Kevin Thies and Vinaya Rao. Work in process pictures follow.
