PORTFOLIO — WEB & INTERACTIVE

Interrelationships Projection

This is an interactive experience that I created in the fall of 2017. Participants fill out a questionaire regarding their own living spaces and then enter someone else's space to find their responses projected over the unfamiliar space. The web app was made with js and php on the server side. The code is available on GitHub. Check out the full project including the process and technical documentation here.
